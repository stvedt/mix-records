# mix-records
Site is live at: https://mix-records.firebaseapp.com

## Setup
To install Node Dependencies for Grunt:

`npm install .`

## Build Static Files
We are using Grunt Includes to pull in the header and footer for better maintainability.

Run `grunt`

## Deploy
````
grunt
firebase deploy
````
